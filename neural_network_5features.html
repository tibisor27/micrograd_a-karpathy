Input Layer (5 noduri)

Fiecare nod = o feature de intrare. Dacă ai block_size=5 cu n_embed=1, sau 5 features numerice directe.

Hidden Layer (200 noduri)

Neuronii ascunși cu activare tanh(). Fiecare primește input de la TOATE cele 5 noduri — fully connected.

Output Layer (27 noduri)

Un nod per caracter din vocabular (a-z + '.'). Softmax convertește în probabilități.

Conexiuni (Weights)

Linii = weights. Input→Hidden: 5×200 = 1000. Hidden→Output: 200×27 = 5400. Total weights: 6400.

📐 Calculul Matematic (Forward Pass)

h = tanh(X @ W1 + b1)    → shape: (batch, 200)
logits = h @ W2 + b2    → shape: (batch, 27)
probs = softmax(logits)    → probabilitate per caracter
